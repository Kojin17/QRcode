# Генератор QR кодов на Python с использованием Tkinter

Этот проект представляет собой простой генератор QR кодов, написанный на языке Python с использованием библиотек Tkinter, qrcode и Pillow.

## Особенности

- Создание QR кода из введенного текста
- Отображение сгенерированного QR кода
- Возможность сохранения QR кода в формате PNG
- Выбор темы оформления для приложения

## Требования к установке

- Python 3.x
- Библиотеки: tkinter, qrcode, Pillow

```bash
pip install qrcode[pil]
```

## Запуск приложения

```bash
python QRcode.py
```

## Использование

1. Запустите приложение.
2. Введите текст в соответствующем поле.
3. Нажмите кнопку "Сгенерировать" для создания QR кода.
4. QR код будет отображен на экране.
5. Нажмите кнопку "Сохранить в PNG", чтобы сохранить QR код в файл.

## Выбор темы

- Нажмите кнопку "Выбрать тему".
- Введите имя темы в диалоговом окне.
- Примените новую тему для изменения внешнего вида приложения.
